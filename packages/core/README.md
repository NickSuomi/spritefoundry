# @nicksuomi/spritefoundry

Effect-first SVG sprite pipeline core for Iconify JSON packages and custom SVG sources.

Install from JSR:

```sh
pnpm add -D jsr:@nicksuomi/spritefoundry
```

Use `buildSpritefoundry` for build orchestration and `createSpriteLoader` for runtime sprite injection.

See the Spritefoundry root README for full config, CLI, Vite, and Vue examples:

https://github.com/NickSuomi/spritefoundry#readme
