# @nicksuomi/spritefoundry-vite

Vite plugin for running Spritefoundry during Vite builds.

Install from JSR:

```sh
pnpm add -D jsr:@nicksuomi/spritefoundry-vite
```

Use `spritefoundryVite` in `vite.config.ts`.

See the Spritefoundry root README for full config and Vite examples:

https://github.com/NickSuomi/spritefoundry#readme
