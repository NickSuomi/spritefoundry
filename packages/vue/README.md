# @nicksuomi/spritefoundry-vue

Vue integration for manifest-backed Spritefoundry icons.

Install from JSR:

```sh
pnpm add jsr:@nicksuomi/spritefoundry-vue
```

Use `createSpritefoundryVue`, `useSpritefoundry`, and `SpriteIcon` in Vue apps.

See the Spritefoundry root README for full Vue examples:

https://github.com/NickSuomi/spritefoundry#readme
